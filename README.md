L7-DC-N.PF.M
===

DevelopmentBase: Laravel7 on docker-compose nginx/php-fpm/mysql

## Setup
1. rm -rf .git && git init
2. cp app/.env.example app/.env
3. docker-compose up

## Versions
- laravel: 7
- Nginx: 1.19
- php-fpm: 7.4
- mysql: 8
